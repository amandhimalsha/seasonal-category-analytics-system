# Seasonal and Category Analytics System

This project is developed for analyzing seasonal and category trends in a modeling/pageant academy using machine learning.

Team Members:
- Project Manager
- Startup Manager
- ML Engineer
- Backend Developer
- Frontend Developer

Tech Stack:
- Frontend: HTML, CSS, JavaScript
- Backend: Python, Flask
- Machine Learning: Python
- Database: MySQL 
- ML & Data Analysis Libraries: Pandas, NumPy, Scikit-learn, Matplotlib/Seaborn
